Bem vindo ao projeto de teste da Lemobs

Este projeto foi desenvolvido como teste para a vaga de desenvolvedor na Lemobs. O projeto foi desenvolvido em Angular 10. O objetivo dele era integrar Views criadas em Tableau para código.

Infelizmente não foi possível utilizar as Views produzidas por falta de acesso ao Tableau Server ou Tableau Online. Foram, então, utilizadas duas Views disponibilizadas na Galeria do Tableau.

As Views produzidas refletem análises realizadas em tabelas de uma base SQL, em PostgreSQL. As Views e o banco de dados estão hospeados no mesmo repositório do GitHub e podem ser acessados clicando aqui.
